- Topic: "{topic}"
- If B2B: prioritize ROI, efficiency, proof. If B2C: prioritize benefits, simplicity, vibe.
- Blog bullets must not repeat the intro. CTA should reference {website_url} if provided, else omit URL.
- Captions per platform (Instagram, TikTok, LinkedIn, X/Twitter, Facebook):

  - TikTok: hook + 1–2 lines; 4–10 hashtags; keep under ~2200 chars.
  - LinkedIn: value-first mini post; 3–5 short lines; no more than 5 tasteful hashtags.
  - X/Twitter: <260 chars; 2–4 hashtags; use abbreviations when natural.
  - Facebook: friendly, 2–3 sentences; 5–10 hashtags max.

OUTPUT
- Strict JSON matching the schema. If unsure, still return valid JSON with best effort.
